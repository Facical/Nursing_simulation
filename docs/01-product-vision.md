# 01. 제품 비전

## 한 줄 비전

> **한국 간호대 학부생이 KABONE 핵심기본간호술 평가를 대비하여 처치실 상황을 3D로 직접 체험하며 술기를 반복 훈련하는 단일 플레이어 데스크톱 시뮬레이터.**

평가 기준은 **한국간호교육평가원(KABONE) 핵심기본간호술 평가항목 프로토콜 제4.1판**에 정렬되며, MVP는 항목 #3 근육주사를 다룬다. 학교 실습실 점유 시간이 제한적이고 실제 환자에게 시도하기 전에 반복 연습할 수단이 부족하다는 문제를 해결한다. 교과서로 암기한 절차를 3D 환경에서 "직접 해보되", 기본 조작은 UI 선택과 손 애니메이션으로 낮추고 세밀 손 조작은 선택형 Hard Hand Simulator로 분리한다.

## 2026-05-25 UX 방향 조정

초기 3D 손 조작 방식은 재미와 몰입감은 있으나, 실제 간호대 학부생의 예습/복습 도구로는 조작 난도가 높다. 따라서 MVP의 기본 경로는 **Basic Simulation Mode**로 고정한다.

- **Basic Simulation Mode (기본값)**: 사용자는 처방 카드, 선택지, 단계별 액션 버튼을 눌러 절차를 진행한다. 손 모델은 해당 액션에 맞는 애니메이션으로 움직이며, 평가는 손 조작 숙련도가 아니라 KABONE 절차 선택·순서·확인 여부에 집중한다.
- **Hard Hand Simulator Mode (설정에서 ON)**: 기존 `ASDF/Space`, `Z/X/C/V/B`, 마우스 그랩/회전 기반 손 조작을 고급 모드로 유지한다. 디버그, 시연, 세밀 조작을 원하는 사용자를 위한 모드이며 MVP 통과의 필수 경로가 아니다.
- **Step 1 UX 우선순위**: 처방 확인 단계는 처방전, 환자 정보, 투약원칙 확인 항목을 카드형 정보 패널로 제공한다.

## 해결하려는 문제

- KABONE 핵심기본간호술 평가는 **단계 누락·핵심항목 통과 실패 시 평가 통과가 어렵지만**, 학생이 집에서 평가 기준 그대로 반복 연습할 수단은 교과서와 영상뿐이다.
- 영상은 관찰 학습은 되지만 **의사결정 훈련**은 불가능하다 (어떤 바늘을 고를지, 어느 랜드마크를 찾을지).
- 실습실 환자 마네킹은 비싸고, 학교 밖에서는 접근이 불가.
- 기존 해외 제품(Practigame 등)은 **한국어 미지원**, KABONE 평가 기준과 불일치.

## 타깃 페르소나

### Persona A. "정민" — 간호대 2학년, 첫 실습 직전
- 다음 주 기본간호학 실습 평가(근육주사)가 있다.
- 교과서를 5번 읽었는데도 순서가 자꾸 헷갈린다.
- 기숙사 노트북(Mac)에서 밤에 혼자 연습하고 싶다.
- **니즈**: 실수해도 처벌 없이 반복할 수 있는 안전한 연습 공간, 한국어 가이드, 실제 체크리스트와 동일한 감점 구조.

### Persona B. "김 교수" — 간호학과 실습 지도교수
- 학생마다 실습실 시간이 부족해 예습·복습 부담을 학생에게 돌리고 싶다.
- 학생들의 사전 학습 도구로 **공인된 절차**를 담은 시뮬레이터를 찾고 있다.
- 장기적으로 수업에서 과제로 내고 디브리핑 리포트를 제출 받고 싶다.
- **니즈**: 교과과정에 부합하는 절차, 디브리핑 리포트 추출, 향후 시나리오 커스터마이징.

## Practigame 대비 차별점 (MVP 범위 기준)

| 항목 | Practigame | 본 프로젝트 MVP |
|---|---|---|
| 언어 | 영어 중심 | **한국어 전용** |
| 플랫폼 | 모바일/웹 중심 | **PC 스탠드얼론** (학교 PC실 배포 용이) |
| 콘텐츠 | 일반 간호 광범위 | **기본간호학 술기 특화**, IM injection 단일 집중 |
| 상호작용 | 탭 기반 | **UI 기반 절차 선택 + 3D 손 애니메이션**, 선택형 Hard Hand Simulator |
| 평가 기준 | 자체 루브릭 | **KABONE 핵심기본간호술 평가항목 프로토콜 제4.1판**과 정합 |
| 아트 방향 | 스타일라이즈드 2D/3D | **Semi-realistic 인물·환경 + 포토리얼 의료 도구** (학습 몰입 충분, uncanny valley 회피) |

## 성공 기준 (MVP)

1. **KABONE 제4.1판 #3 근육주사 21단계가 모두 게임 내 상호작용으로 구현**되고, 핵심항목(*) 6개가 critical gate로 매핑됨.
2. 평가 완료 후 **디브리핑 리포트**가 단계별 결과·총점·감점 사유(KABONE 단계 ID 포함)와 함께 출력됨.
3. 학생 3~5명 비공식 플레이테스트에서 **Basic Simulation Mode 기준** 30분 이내에 한 사이클 완주 가능.

> **참고**: 본 프로젝트는 감수자 검증 단계를 두지 않고 KABONE 공식 PDF 직접 인용으로 의학적 그라운딩을 마감한다. 자세한 결정 배경은 [docs/09-references.md](./09-references.md) 참조.

## 의도적으로 배제하는 것 (Non-goals)

- 멀티플레이어, 강사-학생 실시간 원격 지도
- 모바일/VR 동시 출시 — 집중도 저하 우려
- 생성형 AI 기반 동적 시나리오 — 의학적 정확성 리스크
- 자격증/학점 공식 인증 — 법적·인증 절차 부담, MVP 이후 검토
- 복수 언어 — 1차는 한국어만, 구조만 i18n 가능하도록 남겨둠
