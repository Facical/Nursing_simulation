# 01. 제품 비전

## 한 줄 비전

> **간호대 학부생이 처치실 상황을 3D로 직접 체험하며 기본간호학 술기를 반복 훈련하는 단일 플레이어 데스크톱 시뮬레이터.**

학교 실습실 점유 시간이 제한적이고, 실제 환자에게 시도하기 전에 반복 연습할 수단이 부족하다는 문제를 해결한다. 교과서로 암기한 절차를 3D 환경에서 "직접 해보며" 근육 기억과 임상적 추론을 함께 익히게 한다.

## 해결하려는 문제

- 기본간호학 술기 평가는 **체크리스트 누락 1개만으로도 감점**되지만, 학생이 집에서 복습할 수단은 교과서와 영상뿐이다.
- 영상은 관찰 학습은 되지만 **의사결정 훈련**은 불가능하다 (어떤 바늘을 고를지, 어느 랜드마크를 찾을지).
- 실습실 환자 마네킹은 비싸고, 학교 밖에서는 접근이 불가.
- 기존 해외 제품(Practigame 등)은 **한국어 미지원**, 국내 기본간호학 체크리스트 항목과 불일치.

## 타깃 페르소나

### Persona A. "정민" — 간호대 2학년, 첫 실습 직전
- 다음 주 기본간호학 실습 평가(근육주사)가 있다.
- 교과서를 5번 읽었는데도 순서가 자꾸 헷갈린다.
- 기숙사 노트북(Mac)에서 밤에 혼자 연습하고 싶다.
- **니즈**: 실수해도 처벌 없이 반복할 수 있는 안전한 연습 공간, 한국어 가이드, 실제 체크리스트와 동일한 감점 구조.

### Persona B. "김 교수" — 간호학과 실습 지도교수
- 학생마다 실습실 시간이 부족해 예습·복습 부담을 학생에게 돌리고 싶다.
- 학생들의 사전 학습 도구로 **공인된 절차**를 담은 시뮬레이터를 찾고 있다.
- 장기적으로 수업에서 과제로 내고 디브리핑 리포트를 제출 받고 싶다.
- **니즈**: 교과과정에 부합하는 절차, 디브리핑 리포트 추출, 향후 시나리오 커스터마이징.

## Practigame 대비 차별점 (MVP 범위 기준)

| 항목 | Practigame | 본 프로젝트 MVP |
|---|---|---|
| 언어 | 영어 중심 | **한국어 전용** |
| 플랫폼 | 모바일/웹 중심 | **PC 스탠드얼론** (학교 PC실 배포 용이) |
| 콘텐츠 | 일반 간호 광범위 | **기본간호학 술기 특화**, IM injection 단일 집중 |
| 상호작용 | 탭 기반 | **3D 도구 직접 조작 (drag/click)** |
| 평가 기준 | 자체 루브릭 | **국내 기본간호학 표준 체크리스트**와 정합 |
| 아트 방향 | 스타일라이즈드 2D/3D | **Semi-realistic 인물·환경 + 포토리얼 의료 도구** (학습 몰입 충분, uncanny valley 회피) |

## 성공 기준 (MVP)

1. IM injection 절차의 **모든 체크리스트 항목이 게임 내 상호작용으로 구현**됨.
2. 평가 평가 완료 후 **디브리핑 리포트**가 단계별 결과·총점·감점 사유와 함께 출력됨.
3. 간호학 감수자가 "실습 전 예습 도구로 사용 가능" 판정.
4. 학생 3~5명 비공식 플레이테스트에서 30분 이내에 한 사이클 완주 가능.

## 의도적으로 배제하는 것 (Non-goals)

- 멀티플레이어, 강사-학생 실시간 원격 지도
- 모바일/VR 동시 출시 — 집중도 저하 우려
- 생성형 AI 기반 동적 시나리오 — 의학적 정확성 리스크
- 자격증/학점 공식 인증 — 법적·인증 절차 부담, MVP 이후 검토
- 복수 언어 — 1차는 한국어만, 구조만 i18n 가능하도록 남겨둠
