## 조기 신호와 대응 액션

| 조기 신호 | 이 때 하는 액션 |
|---|---|
| Phase 1 종료 시 1단계도 플레이 불가 | 스코프 절반 축소 (단계 수 7개, 의학적 필수만) |
| 감수자 1차 리뷰에서 절차 지적 >10건 | Phase 2 중단하고 `02-functional-spec.md` 수정 우선 |
| Phase 2 말에 60fps 미달 | URP settings, 텍스처 해상도 일괄 다운, 라이트 Bake |
| 베타 테스터 완주율 < 60% | 조작 튜토리얼 추가, 단계 힌트 기본 활성화 |

## 의학적·법적 주의

- 이 시뮬레이터는 **교육 보조 도구**이며, 실제 환자에 적용 가능한 절차 보증이 아님. 메인 메뉴와 디브리핑에 **면책 문구** 노출.
- 환자 이름·얼굴·병원 상호는 허구. 의심 소지가 있으면 보통명 사용.
- 수집하는 사용자 데이터는 로컬 JSON에 한정. 외부 전송 없음. (향후 LMS 연동 시 개인정보 정책 별도 수립)
