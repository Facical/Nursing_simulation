## 2. 상황 설정 (Briefing)

병동 처치실. 환자는 (예) 통증 호소로 입원 중이며, 의사 처방에 따라 Diclofenac 4mg 1회분을 근육주사로 투여한다. 학생은 간호사 역할이며 처치실에 입장한 직후부터 평가가 시작된다.

### 3.1 처방 확인 (6 rights) — KABONE 단계 2 ★

- **UI**: 전자처방전 창. 체크박스 6개: **대상자 등록번호 / 대상자명 / 약명 / 용량 / 투여경로 / 시간**.
- **합격 조건**: 6개 모두 체크 + 용량 박스에서 처방값(`Diclofenac 4mg`) 직접 선택.
- **실패**: 누락 시 `RequiredItemMissing` 감점, 핵심항목이므로 통과 못하면 critical fail.
- **출처**: `[REF-KABONE-3-2]` "투약처방(투약카드 또는 컴퓨터 출력물 등)과 투약원칙(5 rights; 대상자 등록번호, 대상자명, 약명, 용량, 투여경로, 시간)을 확인한다."

### 3.2 손위생 #1 — KABONE 단계 1

- **UI**: 처치실 싱크대(물·비누). 마우스 클릭/드래그로 비누칠 → 헹구기 애니메이션.
- **합격 조건**: 비누 사용 + 수도꼭지 물 접촉 + 비비기 시간 ≥ 15초.
- **출처**: `[REF-KABONE-3-1]` "물과 비누로 손위생을 실시한다."

> **주의**: 시뮬레이터 손위생은 KABONE에서 1회(#1), 2회(#6 손소독제), 3회(#11 손소독제), 4회(#20 물·비누)로 **총 4회** 분리 평가된다. 1회 누락도 감점 — 단계 5와 9, 12에서 각각 추가 평가됨.

### 3.3 약물 준비 — KABONE 단계 3 ★

- **UI**: 상호작용 줌 카메라. **앰플 절단(또는 절단 보조구) → 앰플에서 주사기로 흡입 → 공기 제거**.
- **합격 조건**:
  - 라벨 확인 (Diclofenac 4mg) 토글
  - 정확한 용량(예: 처방 용량 그대로) 인출
  - 공기 방울 제거
  - 무균술 유지 (앰플 입구 오염 회피)
- **즉시 피드백**: 용량 오차 ±0.1ml 초과 → `DoseOutOfTolerance` 감점, 핵심항목 critical fail 트리거.
- **출처**: `[REF-KABONE-3-3]` "근육주사에 필요한 약물을 정확한 용량 및 방법으로 주사기에 준비한다."

> **변경 사항** (이전 버전 대비): 이전에는 ceftriaxone 바이알 + 재구성(증류수 흡입 → 바이알 주입 → 혼합 → 인출 → 바늘 교체)이었으나 KABONE 표준 약물 Diclofenac 앰플로 변경하여 재구성 단계를 제거했다. 재구성 미니게임은 MVP 후 확장 시나리오로 분리.

### 3.4 물품 준비 — KABONE 단계 4

- **UI**: 트레이 위에 배치할 물품 목록(좌측 UI)과 캐비닛(우측 3D).
- **필수 물품** (`[REF-KABONE-3]` 5. 필요장비/물품 절):
  - 투약카드(또는 컴퓨터 출력물)
  - 일회용 멸균 주사기 2~5cc 규격별 2개씩 (바늘 포함)
  - 소독솜 / 손소독제
  - Diclofenac 4mg 라벨 앰플 2개
  - 투약카트 또는 쟁반(tray)
  - 투약기록지 / 간호기록지
  - 손상성폐기물 전용용기 / 일반 의료폐기물 전용용기
- **유혹 오답(distractor)**: 인슐린 주사기, 26G 피내주사용 바늘, 포비돈 등 → `DistractorItemSelected` 감점.
- **합격 조건**: 필수 물품 전수 + 오답 0개.

### 3.5 환자 접근 + 손위생 #2 + 자기소개 — KABONE 단계 5, 6

- **UI**: 환자에게 접근(이동), 손소독제 펌프(2회) → 비비기 토글, 대화 선택지 "안녕하세요, 간호사 OOO 입니다." 등.
- **합격 조건**: 손소독제 사용 + 자기소개 발화 선택지 클릭.
- **출처**: `[REF-KABONE-3-5]`, `[REF-KABONE-3-6]`.

### 3.6 환자 식별 — KABONE 단계 7 ★

- **UI**: 환자에게 다가가 대화 선택지.
  - A. "김OO님 맞으시죠?" (폐쇄형 단일 확인) → `ClosedQuestionOnly` 감점
  - B. "성함과 생년월일을 말씀해 주시겠어요?" (개방형) → **정답**
  - C. (말 없이 팔찌만 확인) → `OneIdentifierOnly` 감점
- **추가 토글**: 입원팔찌 + 투약카드 대조 (이름 + 등록번호 둘 다).
- **합격 조건**: B 선택 + 팔찌·카드 대조 토글 모두 체크. 둘 중 하나라도 누락 시 critical fail (이름·등록번호 둘 다 확인이 KABONE 명시).
- **출처**: `[REF-KABONE-3-7]` "대상자의 이름을 개방형으로 질문하여 대상자를 확인하고, 입원팔찌와 투약카드를 대조하여 대상자(이름, 등록번호)를 확인한다."

### 3.7 설명 · 동의 · 사생활 — KABONE 단계 8, 9

- **UI**: 대화 선택지 (목적/작용/유의사항 각 1회), 환자에게 의문사항 질문 유도, 커튼 닫기 버튼.
- **합격 조건**: 목적·작용·유의사항 설명 + "의문사항 있으시면 말씀해 주세요" 발화 + 커튼 닫기.
- **출처**: `[REF-KABONE-3-8]`, `[REF-KABONE-3-9]`.

> **변경 사항**: KABONE은 명시적 "동의 획득" 토글을 두지 않는다 — 설명 + 의문사항 질문 유도가 절차상 동의에 해당. 이전 버전의 별도 "동의 획득" 토글은 제거.

### 3.8 부위 선정 + 랜드마크 촉진 — KABONE 단계 10 ★

- **UI**: 환자 모델 주위 후보 부위 ①~④ 하이라이트. MVP는 사례를 **② 둔부 복면**으로 고정.
- **부위 옵션** (`[REF-KABONE-3-10]` 원문):
  - ① 둔부 배면 (dorsogluteal) — 대전자~후상장골극 사선 상외측 / 장골능 5cm 아래 / 둔부 4등분 상외측
  - ② **둔부 복면 (ventrogluteal) — MVP 정답** — 측위, 왼손바닥-대전자, 집게손가락-전상장골극(ASIS), 가운데손가락-장골능 V자
  - ③ 대퇴 — 외측광근 3등분 가운데 / 대퇴직근
  - ④ 삼각근 중앙 — 견봉돌기 5cm 아래
- **랜드마크 촉진** (둔부 복면): 환자 측위 자세 + 마우스 hover로 3개 점 클릭 — **대전자 → 전상장골극 → 장골능** 순.
- **합격 조건**: ② 선택 + 3개 랜드마크 정확한 위치 + 순서 정답.
- **즉시 피드백**: 잘못된 점 클릭 시 빨간 X, 재시도 허용. 부위 ①③④ 선택 시 "사례별 정답 외 부위" 알림 (MVP는 사례 고정이므로 감점이 아니라 sample mismatch).
- **출처**: `[REF-KABONE-3-10]`.

> **삭제 사항** (이전 버전 대비): "삼각근은 부피 초과(>2ml)로 감점", "둔부 배면은 좌골신경 리스크로 감점(권장 안 함)" 규칙은 **KABONE 4.1판 프로토콜에 없으므로 삭제**. 임상 best-practice이지만 KABONE 평가 기준과 별개.

### 3.9 손위생 #3 + 피부 소독 — KABONE 단계 11, 12

- **UI**: 손소독제 펌프(2회) + 비비기 토글 → 알콜솜으로 부위 소독.
- **합격 조건**:
  - 손소독제 사용
  - 알콜솜 **안쪽 → 바깥쪽** 원형 (직경 5~8cm)
  - 마름 대기 후 약 라벨 한 번 더 확인 + 바늘뚜껑 제거
- **즉시 피드백**: 외→내 역방향, 지그재그 → `DisinfectionPathWrong` 감점.
- **출처**: `[REF-KABONE-3-11]`, `[REF-KABONE-3-12]`.

### 3.10 주사 시퀀스 — KABONE 단계 13, 14 ★ (둘 다 핵심)

- **UI**: 상호작용 줌 + 각도 인디케이터.
- **하위 시퀀스**:
  1. 한 손으로 피부 팽팽하게 잡기
  2. **주사바늘 90° 유지** + 빠르게 삽입 (`[REF-KABONE-3-13]`)
  3. 잡았던 손의 엄지·집게로 주사기 하단부 잡기, 다른 손으로 **내관 살짝 뒤로 당겨 흡인** (`[REF-KABONE-3-14]`)
  4. 혈액 없음 확인 → 천천히 약물 주입
  5. (혈액 보이면 주사기를 빼고 처음부터 다시 — KABONE 명시)
- **합격 조건**: 90° ±10°, 흡인 수행, 혈액 없음 분기 정답 처리, 주입 속도가 인디케이터 범위 내.
- **즉시 피드백**: 각도 이탈, 흡인 누락 → 핵심항목 critical fail.
- **출처**: `[REF-KABONE-3-13]` "주사바늘을 90°로 유지한 다음 주사기로 선정된 주사부위 근육을 재빨리 찌른다.", `[REF-KABONE-3-14]` "...주사기의 내관을 살짝 뒤로 당겨 혈액이 나오지 않으면 ... 천천히 주입한다. (만약 주사기에 혈액이 보인다면 주사기를 빼내어 버린 다음 주사 준비를 처음부터 다시 해야 함)".

> **삭제 사항**: "Z-track 기법(2~3cm 옆으로 당기기)", "Z-track 시 흡인 생략" 규칙은 **KABONE 4.1판에 없으므로 삭제**.

### 3.11 발침 · 마사지 · 자세 · 후 설명 · 커튼 걷기 — KABONE 단계 15, 16, 17, 18

- **UI**: 발침 버튼(삽입 각도 그대로), 소독솜으로 누르며 마사지 토글, 자세 편안하게(환의 입히기), 후 설명 발화 선택지, 커튼 걷기 버튼.
- **합격 조건**:
  - 삽입 각도 그대로 발침 (`AngleOutOfRange` 감점 가능)
  - 소독솜으로 누르며 마사지 (출혈 시 1~2분 압박)
  - 환의 + 자세 편안하게
  - 후 기대효과 설명 발화
  - 커튼 걷기

### 3.12 폐기 + 손위생 #4 + 기록 — KABONE 단계 19, 20, 21

- **UI**: 사용한 주사기/바늘을 드래그하여 폐기. 손위생(물·비누) 토글. 기록지 작성 폼.
- **합격 조건**:
  - **바늘 캡 씌우지 않고** 손상성폐기물 전용용기에 폐기 (`NeedleRecapped` 감점)
  - 사용한 소독솜·주사기는 일반 의료폐기물 전용용기에 폐기 (`SharpsBinMissed` 감점 — 잘못된 용기 선택)

## 4. 평가 및 디브리핑

### 4.1 점수 체계

- 시뮬레이터 가중치 합 100점 (위 매핑 표 참조).
- 단계 내부는 감점식: 최대 가중치에서 실수별로 차감, 0점 하한.
- **Critical fail (핵심항목 통과 실패)**: 해당 step 점수 0점 처리 + 디브리핑 경고 배지. 핵심항목은 KABONE에서 명시한 6개 항목 — 단계 1, 3, 6, 8, 10 (10에 KABONE 13/14 두 핵심 묶음).
- **종합 critical fail 사유** (전체 시나리오 실격성):
  - 환자 식별 실패 (1 identifier만)
  - 흡인 누락 또는 혈액 보임에도 주입 강행
  - 무균술 파손 (앰플/바늘 오염 후 그대로 사용)
  - 바늘 재캡 후 손상성폐기물에 투기

> **KABONE 평가와의 차이**: KABONE 실제 평가는 "수행/미수행" 또는 등급식이고 우리 시뮬레이터는 학습 효과를 위해 100점 점수식을 사용한다. 시뮬레이터 점수는 KABONE 통과 가능성의 근사치이며, 핵심항목 critical fail이 발생하면 실제 평가에서는 통과 어렵다는 점을 디브리핑에 명시한다.

### 4.2 디브리핑 리포트 구성

1. **요약**: 총점, 소요시간, 핵심항목(★) 통과 6/6 여부, critical fail 건수
2. **단계별 결과표**: 시뮬레이터 단계명 + KABONE 단계 ID 매핑 / 획득 / 만점 / 주요 감점 사유 (KABONE 원문 요약 포함)
3. **학습 포인트**: 감점 상위 3개 항목 — KABONE 프로토콜 원문 한 줄 인용으로 표시
4. **재도전** 버튼 / **메인 메뉴** 버튼 / (추후) **CSV 내보내기**

### 4.3 피드백 타이밍 정책

- **즉시 피드백**: 환자 안전 직결(손위생, 환자 식별, 용량 오류, 흡인 누락) → 빨간 토스트 + 재시도 허용.
- **지연 피드백**: 절차적 실수(설명 선택지 누락, 자세 토글 미체크 등) → 디브리핑에서만 공개. 실전 감각 유지 목적.
