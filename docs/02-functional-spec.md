# 02. 기능 명세 — IM Injection (근육주사)

> **주의**: 본 문서의 술기 절차는 국내 기본간호학 표준 교재(양선희 외 *기본간호학*, 현문사 등)와 대한간호협회 실습 체크리스트를 **참조해 초안으로 작성**한 것이다. 최종 구현 전 반드시 **간호학 감수자 리뷰**를 거쳐 확정할 것 (마일스톤 R2 참조).

## 1. 시나리오 개요

| 항목 | 값 |
|---|---|
| 시나리오 ID | `SCN_IM_INJECTION_001` |
| 제목 | 근육주사 투약 (IM Injection) |
| 난이도 | 기본 (Level 1) |
| 예상 플레이 시간 | 10 ~ 20 분 (1회 사이클) |
| 대상 환자 | 성인 남성(가상), 앉은 자세 또는 앙와위 |
| 처방 예시 | `ceftriaxone 1g IM q12h` (바이알 재구성 포함) |

## 2. 상황 설정 (Briefing)

병동 처치실. 환자는 폐렴 진단으로 입원 중이며, 의사 처방에 따라 항생제 1회분을 근육주사로 투여해야 한다. 학생은 간호사 역할이며, 이미 손씻기·환자 인수인계를 마친 상태에서 처치실에 입장한다.

## 3. 절차 단계 (Step List)

각 단계는 [05-data-model.md](./05-data-model.md)의 `ScenarioStep` 하위 타입 중 하나로 구현된다. 아래 표의 **Step Type**이 그 매핑이다.

| 3 | 물품 준비 | ToolInteraction | 10 | 지연 |

### 3.1 처방 확인 (5 Rights)
- **UI**: 전자처방전 창이 뜨고, 체크박스 5개: 환자(Right Patient), 약물(Drug), 용량(Dose), 경로(Route), 시간(Time).
- **합격 조건**: 5개 모두 체크, 그리고 용량 박스에서 처방된 `1g`을 직접 드롭다운에서 선택.
- **실패 분기**: 체크 없이 Next 클릭 시 감점 2점 + 힌트 팝업 (지연 피드백으로 디브리핑에 기록).

### 3.2 손위생
- **UI**: 처치실 싱크대 or 손소독제 펌프. 마우스 클릭/드래그로 소독제 펌프 → 양손 프리팹에 비벼지는 애니메이션 3~5초.
- **합격 조건**: 소독제 펌프 2회 이상 + 15초 이상 비비기.
- **즉시 피드백**: 이 단계를 건너뛰고 3번으로 진행 시도 → 화면 상단 경고 "손위생이 필요합니다".

### 3.3 물품 준비
- **UI**: 트레이 위에 배치해야 할 물품 목록(좌측 UI)과 캐비닛(우측 3D).
- **필수 물품**: 주사기 3cc, 바늘 23G×1인치(성인 IM 기준), 21G 인출용 바늘, 알콜솜 2개, 바이알(항생제), 증류수 앰플, 장갑, 샤프스 컨테이너, 트레이, 처방전.
- **유혹 오답(distractor)**: 인슐린 주사기, 26G 피내주사용 바늘, 포비돈 — 선택 시 감점.
- **합격 조건**: 필수 물품 전수 + 오답 물품 0개.

### 3.4 약물 재구성 & 용량 인출
- **UI**: 상호작용 줌 카메라. 바이알 → 앰플에서 증류수 흡입 → 바이알 주입 → 혼합(흔들기/굴리기) → 주사기로 용량 인출 → 공기 제거.
- **합격 조건**:
  - 인출용 바늘(21G) 사용
  - 바이알 상부 알콜 소독
  - 정확히 3.5ml(예시 재구성 비율) 인출
  - 공기 방울 제거 완료
  - 인출 후 바늘을 주사용 23G로 교체
- **즉시 피드백**: 용량 오차 ±0.2ml 초과 시 빨간 피드백 + 재시도 허용.

### 3.5 환자 확인
- **UI**: 환자에게 접근 후 대화 선택지.
  - A. "김OO님 맞으시죠?" (폐쇄형 단일 확인 — 감점)
  - B. "성함과 생년월일을 말씀해 주시겠어요?" (개방형, 2 identifier — **정답**)
  - C. (아무 말 없이 팔찌만 확인 — 감점)
- **합격 조건**: B 선택 + 팔찌 바코드/팔찌 확인 토글 체크.

### 3.6 설명 · 동의 · 프라이버시
- **UI**: 대화 선택지 3개(설명 문장), 커튼 닫기 버튼(프라이버시), 동의 획득 토글.
- **합격 조건**: 목적·부작용·절차 설명 선택지 각각 1회 + 커튼 닫기 + 동의 획득 체크.

### 3.7 주사 부위 선정
- **UI**: 환자 모델 주위 4개 후보 부위에 하이라이트: 삼각근(deltoid), 외측광근(vastus lateralis), 둔부복면(ventrogluteal), 둔부배면(dorsogluteal).
- **합격 조건**: 환자 상태(성인/근육량 양호)와 약물 부피(3.5ml)에 맞는 부위 선택 → **외측광근 또는 둔부복면**이 정답. 삼각근은 부피 초과(>2ml)로 감점, 둔부배면은 좌골신경 리스크로 감점(권장 안 함).

### 3.8 랜드마크 촉진
- **UI**: 선택된 부위에서 뼈 랜드마크 포인트 3개가 숨겨져 있음. 마우스 hover로 찾고 클릭 순서대로 짚기.
  - 둔부복면 예: 대전자 → 전상장골극 → 장골능선 세 점.
  - 외측광근 예: 대전자 → 슬개골 사이 중앙 1/3 지점.
- **즉시 피드백**: 잘못된 점 클릭 시 빨간 X, 재시도 허용.

### 3.9 피부 소독
- **UI**: 알콜솜을 커서로 드래그. 선택 부위 중심에서 원형 나선으로 외곽까지.
- **합격 조건**: 중심→외곽, 원형 경로 90% 이상 커버, 드라이 타임 10초 대기 토글.
- **즉시 피드백**: 외곽→중심 역방향, 지그재그 드래그는 경고.

### 3.10 주사 · 흡인 · 주입 · 발침 (Sequence)
- **UI**: 상호작용 줌 + 각도/속도 미니 인디케이터.
- **하위 스텝**:
  1. 한 손으로 피부 팽팽하게 잡기 (또는 Z-track: 2~3cm 옆으로 당기기)
  2. 주사기 90°로 빠르게 삽입
  3. (흡인) 기관 지침에 따라 선택. Z-track 시 생략이 표준.
  4. 약물 주입 속도: **1ml/10초** 수준 느리게 (인디케이터 범위 내)
  5. 10초 유지 후 들어간 각도 그대로 발침
  6. Z-track이면 당긴 피부 놓기
- **즉시 피드백**: 각도 편차 ±10°, 주입 속도 범위 이탈, 발침 각도 편차 → 감점.

### 3.11 압박 · 폐기 · 기록 · 관찰
- 거즈로 가볍게 압박 (문지르지 않기 — 문지르면 감점)
- 사용 바늘 **캡 씌우지 않고** 샤프스 컨테이너 바로 투기 (캡 씌우면 "재캡 금지" 감점)
- 투약 기록지 작성: 날짜/시간/약물/용량/경로/부위/서명 6개 필드
- 5분 후 부작용 관찰 (시계 아이콘 skip 버튼 — 관찰 skip 시 감점)

## 4. 평가 및 디브리핑

### 4.1 점수 체계
- 각 단계 가중치 합 100점.
- 단계 내부는 감점식: 최대 가중치에서 실수별로 차감, 0점 하한.
- 치명적 실수(critical fail)는 단계 점수 0점 처리 + 디브리핑에서 경고 배지:
  - 손위생 누락
  - 잘못된 환자 확인(1 identifier만)
  - Z-track 없이 둔부배면 선택 + 좌골신경 주변 랜드마크
  - 오염된 바늘 재사용

### 4.2 디브리핑 리포트 구성
1. **요약**: 총점, 소요시간, 치명 실수 건수
2. **단계별 결과표**: 단계명 / 획득 / 만점 / 주요 감점 사유
3. **학습 포인트**: 감점 상위 3개 항목에 대해 교과서 근거 문구 표시 (짧은 이론 링크)
4. **재도전** 버튼 / **메인 메뉴** 버튼 / (추후) **CSV 내보내기**

### 4.3 피드백 타이밍 정책
- **즉시 피드백**: 환자 안전에 직결되는 실수(손위생, 환자 식별, 용량 오류) → 빨간 토스트 + 재시도 허용.
- **지연 피드백**: 절차적 실수(기록 누락, 물품 준비 사소 누락) → 디브리핑에서만 공개. 실전 감각 유지 목적.
